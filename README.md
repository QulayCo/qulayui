<h2 align="center">
  QulayUI
</h2>

<p align="center">
Completely unstyled, fully accessible UI components, designed to integrate beautifully with Tailwind CSS. 
UI components for Vue 3
</p>

## Installation

Please note that **this library only supports Vue 3**.

```sh
# npm
npm install qulayui

# Yarn
yarn add qulayui
```
